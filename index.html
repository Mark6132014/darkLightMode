Hello! Isn't this confusing??

This is another illusion. This is actually in dark mode. Didn't you know that? But the background is white.

How to do it?

HTML Code:

<meta name="color-scheme" content="dark">

CSS Code:

html, body * {
   background: white;
   color: black;
}

How to understand this is really in dark mode?

Well you see, when you look closely at the scrollbar, you will see the scrollbar dark.
Which means that the dark mode meta tag is added to the webpage, or using scrollbar-color in CSS.
You can only notice it on Desktops. Not mobile devices!
